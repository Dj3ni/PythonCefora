# Créer de comptes courants

Attributs
 - numero
 - titulaire: Personne
 - solde

Comportement:
 - Retrait(somme)
 - Depot(somme)

Contrainte:
 - Pas de retrait de somme négative
 - Pas de dépot de somme négative
 - Retrait ne doit amener à un solde négatif
